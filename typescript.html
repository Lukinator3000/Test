TypeScript Tips

TypeScript is a programming language that extends upon JavaScript with improved bug reducing features. Namely, TypeScript uses static typing so that the type of variables cannot be changed after they're initialized, preventing many bugs associated with JavaScript's dynamic typing. Also unlike JavaScript, browsers don't understand TypeScript, so TypeScript files need to be compiled into JavaScript files.
